WEB2-Domain Name System-1.수업소개

host - 인터넷(네트워크)에 연결된 컴퓨터 한대한대를 host라고한다

host와 host가 서로 통신을 하기 위해서는 서로에 대한 주소가 필요합니다. 인류에서 사용하는 주소를 ip address라고 합니다.

ip address 덕분에 인류는 host와 host를 연결해서 통신을 할 수 있게 되었고 이것은 한대의 컴퓨터가 한대의 한계를 추월해버리는 위대한 순간이다.

WEB2-Domain Name System-10.nslookup

nslookup '도메인' 로 터미널에서 검색 가능(권한 없는 IP 추출)

nslookup -type=ns '도메인' 로 터미널에서 검색 가능 (NS 추출)

nslookup '도메인' '네임서버(NS)' 로 터미널에서 검색 가능 (IP 추출)
