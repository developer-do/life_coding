backup

데이터를 백업하자. 모으는 거만 못할수가 있다.

백업의 기본적인 원리는 데이터를 복제해서 보관하자. 이다.

내 컴퓨터와 별도의 컴퓨터에 데이터를 복제해서 보관한다면 데이터를 동시에 잃어버릴 가능성은 현저히 낮아진다.

그 컴퓨터가 독립된 건물에 떨어져 있다면 훨씬더 유실될 가능성은 떨어집니다.

그리고 그 건물들이 다른 국가에 있다, 행성에 있다. 하면 더욱 더 유실될 가능성은 떨어진다.

backup에 찾아봐야할 keyword는 mysqldump / binary log 등등등


cloud

cloud란 컴퓨터를 데이터베이스 서버로 이용하는것이 아니고 엄청나게 큰(마소, Google, apple, amazon)회사들이 운영하고 있는 인프라 위에 있는 컴퓨터를 임대해서 사용하는 것이 클라우드 컴퓨팅 라고 한다.

그래서 컴퓨터가 나한테 있지 않다. 그 회사들의 데이터 센터에 있고 그 컴퓨터를 우리는 눈으로 볼 수 없다. 다루는 방법은 원격 방법이다.

cloud의 keyword는 AWS RDS, Google Cloud SQL for MySQL, AZURE Database for MySQL 등등등
